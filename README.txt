1) Öffnen sie eine Commandozeile
2) Prüfen sie mit 'node --version' und 'npm --version' ob NodeJS insatlliert ist.
	Wenn nicht muss NodeJs für ihr Betriebssystem Installiert werden (https://nodejs.org/en/download/)
	führen sie danach im Terminal 'npm install' aus
3) navigieren sie im Terminal in den Ordner des Repositories (RSS-FEED-5127548) 
4) starten sie 'npm install node-modules' und danach 'npm install -g @angular/cli' 
um die nötigen files und angular-strukturen zu laden

5)öffnen sie ein zweites Terminal und navigieren sie in den Ordner des Repositories/server (RSS-FEED-5127548/server)
6) führen sie 'npm install node-modules' aus und npm install ibm-watson'
7) starten sie in diesem Terminal mit 'node server.ts' den Webserver auf localhost:5200
-> Der Server läuft! Hurra!! Wechseln sie wieder in das erste Terminal
8) starten sie mit 'ng s' dias Client-Seitige Frontend auf localhost:4200
-> Das Frontend läuft. Viel Spaß bei der benutzung
